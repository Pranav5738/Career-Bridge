# 🚀 Carrier Bridge — AI Career Platform

- AI Resume Analyzer (TF-IDF + Cosine Similarity)
- Mentor Marketplace (Booking + Save)
- Mock Interview System
- Webinar Management
- Community Forum
- JWT Authentication

- Resume vs Job Description matching
- Keyword gap analysis
- Smart suggestions

- MongoDB

AI:
- Natural (NLP)
- TF-IDF
- Cosine Similarity
